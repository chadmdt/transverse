Trois dossiers sont nécessaires : 
- Un qui contient "main.py" et "slice.pml"
- Un qui contiendra les photos (doit être vide au début)
- Un qui contient les fichiers pdb à traiter (sera vide à la fin, les fichiers
seront supprimés au fur et à mesure)
Nes changer les adresses que dans la fonction prog

Ouvrir PyMOL, déplacer le Working Directory au dossier qui contient main.py
Commande PyMOL (dans le terminal de PyMOL) : run main.py

Cycle à reproduire jusqu'à ce que soit affiché 'ERROR: The folder is empty'
Commande PyMOL (dans le terminal de PyMOL) : prog
Cliquer sur 'ramp1010RWB' dans les objets pour la faire disparaitre (facultatif)
Commande PyMOL (dans le terminal de PyMOL) : draw 100,100
(ou autre taille)
